# Video Website
Here is an alternative video viewer that could be used for 3 videos. 

## Setup
* Download [Python](https://www.python.org/downloads/) (You should already have it downloaded).

* Go into the video-viewer-website folder and open the command prompt. 

* Run the following command: 
```
python -m http.server
```

* Go to localhost:8000

* Put your browser in full screen mode

* Have Fun!!!
